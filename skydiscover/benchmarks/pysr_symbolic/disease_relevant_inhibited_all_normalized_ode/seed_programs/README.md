# Inhibited ODE Seed Programs

This directory contains 34 structurally distinct iteration-0 programs. They are deliberately different kinetic families rather than parameter or sign variants of one capacity-limited template.

The population spans autonomous and non-autonomous dynamics; monotone, overshooting, reversible, and declining trajectories; logistic, Gompertz, Richards, Finke-Watzky, Hill, Michaelis-Menten, Langmuir, saturation, variable-order, transient-pulse, moving-equilibrium, oscillatory, and signed-flux forms.

The experiment runner defaults `SKYDISCOVER_SEED_TOP_K=all`, ensuring fast-pass ranking does not discard whole structural families before AdaEvolve starts.
