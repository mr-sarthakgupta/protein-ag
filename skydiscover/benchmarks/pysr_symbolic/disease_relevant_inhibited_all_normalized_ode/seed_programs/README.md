# Inhibited ODE Seed Programs

This directory contains 79 additional single-equation seed programs. Together with `../initial_program.py`, AdaEvolve starts from 80 diverse iteration-0 programs distributed round-robin across the four islands.

The variants deliberately cover autonomous kinetics, time-gated kinetics, inhibitor denominators, monomer sequestration, state-dependent inhibition, Gompertz-like capacity terms, terminal-slope shaping, and channel-specific suppression.
